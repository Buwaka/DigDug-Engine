Vrij liberale interpretatie van DigDug.
Je shit je pomp om de monsters te doden, hierdoor krijg je score, dood ze allemaal en je mag door naar het volgende level.
Het volgende level heeft meer enemies, ze blijven opstapelen!
Kan ook tot 4 mensen co-op gespeelt worden, dit doe je door het aantal spelers in controlers aan te sluiten voordat je de game opent.
De UI bestaat uit 3 elementen: Level, Lives en Score (in die volgorde).
Wanneer je al je levens verliest ga je een level terug naar onder, eindig je op level 0 is het game over.
Probeer tot level 10 te geraken!

Player 1 kan bestuurd worden door keyboard, rest is controller
Arrow keys : Movement
Space: Shoot
ESC or F4: Quit Game

Controller:
Dpad or Left Thumbstick: Movement
X or A: Shoot
